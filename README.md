# Create Custom Watson Studio Projects Dynamically Using the Watson Data API

##### By Jeanette Pranin

This repository includes a Postman collection containing sample requests detailed in the blog post "Create Custom Watson Studio Projects Dynamically Using the Watson Data API" (url here). This collection contains examples of how to structure your requests to the Watson Data API, including:

1. Generating an IAM bearer token
2. Creating a Watson Studio Project
3. Creating a Database Connection
4. Adding Members

*Please note that some values must be updated with the appropriate information for the postman collection to run.*
